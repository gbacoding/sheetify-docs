# Sheetify — Public Documentation

Sheetify is a B2B SaaS designed to automate the extraction and transformation of invoice data into clean, structured Excel spreadsheets.  
Its goal is to eliminate manual data entry, reduce operational errors, and streamline financial workflows for companies that manage large volumes of invoices.

This repository contains the **public documentation** of the project.  
It does **not** include the application’s source code, which is stored in a private repository.
